# IP Invs

## For Linux

1. Download inv.sh

```sh
 curl -O http://YOURAPPURL/inv.sh
```

2. Run inv.sh

```sh
  sh inv.sh
```

One action
```sh
 curl http://YOURAPPURL/inv.sh | sh
```


## For Windows

1. Download inv.vbs
```ps1
 PS> Invoke-Restmethod http://YOURAPPURL/inv.vbs < inv.vbs
 ```
 
2. Run inv.vbs

  ```cmd
  CMD> cscript inv.vbs
  ```
